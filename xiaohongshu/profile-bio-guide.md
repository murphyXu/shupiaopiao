# 小红书主页简介 · 合规版

## 原简介为何违规

```
童书绘本闲置漂流｜同名公众号看操作指南  ❌
```

触发点：
- **「公众号」** → 明示第三方平台，属于站外导流
- **「看操作指南」** → 诱导用户离开小红书
- **「同名」** → 暗示跨平台搜索，审核易判营销号

小红书简介禁止：微信号/手机号/二维码、公众号/小程序/抖音等外链平台名、谐音变体（薇、V❤）、「关注领取」「私信获取」等诱导话术。

## 推荐简介（按安全程度排序）

### ⭐ 首选 · 纯定位（最安全，建议先用这条）

```
童书绘本共读日常｜书架整理·分龄书单·闲置怎么处理
```

### 备选 A · 人设向

```
三岁娃妈｜记录在养的书架与绘本清单
```

### 备选 B · 品牌向（不出现「漂流」营销感）

```
让童书被多读几次｜亲子阅读与绘本整理分享
```

### 备选 C · 可配合置顶笔记（仍不写站外平台）

```
绘本整理进行中｜分龄书单与闲置处理经验｜置顶有汇总
```

说明：「置顶有汇总」是**站内跳转**，比写公众号合规得多。

## 不建议再用的写法

| 写法 | 原因 |
|------|------|
| 同名公众号 / 微信搜 xxx | 站外平台名 |
| 小程序书漂漂 | 外链产品导流 |
| 私信领指南 / 关注领资料 | 诱导互动 |
| 薇信 / 公粽号 / gzh | 变体一样抓 |
| 邮费到付免费领书 | 易被判定营销 |

## 合规引流放在哪（不写进简介）

| 位置 | 做法 |
|------|------|
| **置顶笔记** | 纯干货《闲置绘本怎么处理》最后一页写「完整步骤见主页置顶」——**不要**在置顶里放二维码/微信号；站外引导放评论区自评，且等用户主动问再回 |
| **笔记正文** | 价值内容为主；文末用「整理成清单了」代替「去公众号看」 |
| **评论区** | 仅当用户问「怎么用」时回复，避免主动发「搜公众号」；可说「我主页置顶有整理」 |
| **粉丝群** | 开通后放群公告（平台内相对安全） |
| **企业号** | 可挂官方咨询组件（需认证） |
