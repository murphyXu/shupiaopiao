# 真实截图指南 · 2026-07-02

## 通用步骤（微信开发者工具）

1. 打开 **微信开发者工具** → 导入项目 `shupiaopiao-cloud`
2. 确认 `miniprogram/config/index.js` 云环境已配置
3. 编译运行 → 模拟器选 **iPhone 14**（390×844）
4. 截图方式（任选）：
   - macOS：`Cmd + Shift + 4` 框选模拟器区域
   - 或开发者工具菜单 **工具 → 截取屏幕**
5. 将截图保存到本目录 `images/`，**覆盖**下表对应文件名
6. 如有 `无`，运行合成命令

## 打码要求

| 信息 | 处理 |

| 积分余额（可选） | 可保留或打码 |
