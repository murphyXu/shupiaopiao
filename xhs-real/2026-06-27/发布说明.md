# 小红书 · 2026-06-27（真人实拍版）

全新独立文件夹 `xhs-real/`，与旧 `xiaohongshu/` 无关。
定位：真实妈妈分享，不堆营销话术，图片全部自己拍。

## 文件
| 文件 | 用途 |
|---|---|
| `title.txt` | 标题（口语，带省略号那种随手感） |
| `body.txt` | 正文（生活流水账口吻，结尾抛问题互动） |
| `tags.txt` | 话题标签 |
| `comment-followup.txt` | 发布后评论区补充 |
| `拍摄参考.md` | ⭐ 逐张教你怎么拍 |
| `images/` | 放你拍好的照片（空文件夹，自己填） |

## 发布步骤
1. 按 `拍摄参考.md` 拍 4-6 张照片，放进 `images/`
2. 标题复制 `title.txt`，正文复制 `body.txt`，话题贴 `tags.txt`
3. 封面那张在小红书 App 里加一行口语字幕（别做成海报）
4. 发布后有人问，评论区回 `comment-followup.txt`

## 口吻自检（发之前读一遍正文）
- [ ] 读起来像跟朋友聊天，不像产品文案
- [ ] 有具体细节（数量、家里哪个角落、漂了几本）
- [ ] 没有「最/必入/强烈推荐」这类词
- [ ] 没出现公众号/微信/二维码字样
- [ ] 邮费写的是「接书的人到付」，没承诺包邮
- [ ] 图片是自己拍的真实照片 + 1 张小程序截图，不用 AI 图
