# 书漂漂公众号 & 小红书引流运营方案

**日期**：2026-06-26  
**状态**：已确认，待执行

## 目标

通过同名微信公众号（信任中枢 + 转化）与小红书（公域种草）为「书漂漂」微信小程序引流，聚焦童书绘本闲置漂流场景，30 天内建立稳定内容节奏与可追踪转化链路。

## 平台分工

| 平台 | 角色 | 核心任务 |
|------|------|----------|
| 小红书 | 公域获客 | 育儿痛点 + 真实体验，导关注同名公众号 |
| 公众号 | 信任沉淀 + 转化 | 操作指南、案例故事，菜单/推文导小程序 |
| 小程序 | 成交阵地 | 漂流广场浏览 → 接漂 / 上漂 |

## 最优引流链路

```
小红书种草 → 搜「书漂漂」公众号 → 菜单/推文小程序卡片 → 小程序漂流广场
```

**禁止**：小红书直接放微信号、小程序码、二维码（易限流）。  
**允许**：简介与文末引导「微信搜同名公众号」。

## 合规红线

与产品 `scripts/test-compliance-p0-contract.js` 及 `pointRules.js` 一致：

- 不说「邀请送积分」「关注领奖励」等诱导话术
- 公益积分表述为「站内流转工具」，不可提现、不可交易
- 不用「钱包」「返现」「赚钱」
- 接漂说明「邮费到付」，不承诺「完全免费包邮」
- 不使用绝对化用语（最强、必入、100% 等）

## 内容矩阵（长期）

| 栏目 | 占比 | 平台 |
|------|------|------|
| 痛点共鸣 | 30% | 小红书为主 |
| 操作指南 | 25% | 公众号为主 |
| 真实案例 | 25% | 双平台 |
| 绘本轻推荐 | 20% | 小红书为主 |

**发布节奏**：小红书 3–4 篇/周；公众号 2 篇/周。

## 账号基建

### 公众号

- 头像/简介与小程序一致；简介含「童书绘本免费漂流，邮费到付」
- 菜单：开始漂流（小程序）｜新手指南｜常见问题
- 关键词：漂流、接书、上漂、积分、指南
- 每篇推文文末固定小程序卡片 + 行动号召

### 小红书

- 昵称/头像统一「书漂漂」
- 简介：「童书绘本闲置漂流｜同名公众号看操作指南」
- 置顶：《书漂漂是什么？3 步完成童书漂流》
